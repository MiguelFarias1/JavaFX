Instalação e Configuração do Ambiente JavaFX

O JavaFx é muito amplo e pode-se fazer o uso de diversas ferramentas e recursos para desenvolvê-lo, porém nós utilizaremos o IntelliJ IDE e o Scene Builder como principais plataformas para estarmos desenvolvendo este tutorial.

Veja o Tutorial de instalação dessas ferramentas tanto para Windows como para Linux, logo a seguir:

Instalação para Windows

Instalação do IntelliJ

Passo 1: Na barra de pesquisa do seu navegador digite "IntelliJ", faça a busca e clique na primeira página cujo o domínio seja de "jetbrains.com".



Passo 2: Clique no botão "Baixar" no canto inferior esquerdo.



Passo 3: Na aba de Download, clique em "Baixar" logo abaixo de "Community Edition".



Passo 4: Espere a conclusão do download e clique no arquivo baixado.



Passo 5: Ao abrir o instalador, clique em "Next", selecione o local a ser instalado e clique novamente em "Next".



Passo 6: Selecione a caixa "Java" que é a linguagem que utilizaremos, a caixa "Update PATH Variable", pois podem ocorrer alguns erros caso ela não seja marcada e selecione "IntelliJ IDE" caso você queira criar um ícone na área de trabalho. Logo após, clique em "Next".



Passo 7: Clique em "Install" e espere a conclusão da instalação.



Passo 8: Após a conclusão da instalação, selecione a opção "Reboot now" para reiniciar sua máquina, clique em "Finish" e Pronto!! Sua instalação do IntelliJ está concluida.





Instalação do Scene Builder

Passo 1: Na barra de pesquisa do seu navegador digite "JavaFX", faça a busca e clique na página "JavaFX" com o dominio "openjfx.io".



Passo 2: Clique no botão "Download" logo abaixo de Scene Builder. Logo após você será direcionado para outra página.



Passo 3: Ao lado direito da página, clique no botão verde de "Download Now".



Passo 4: Escolha a opção de download para Windows, clique em "Download" e aguarde a conclusão do download.



Passo 5: Após a conclusão do download clique para abrir o arquivo baixado.



Passo 6: Após abrir o arquivo clique em "Next".



Passo 7: Selecione a opção que concorda com os termos de licença e clique novamente em "Next".



Passo 8: Selecione a pasta onde o javaFX será instalado, clique em "Next" e aguarde o processo de instalação.



Passo 9: Após o processo de instalação ser concluido, clique em "Finish" e pronto!! Sua instalação do JavaFX com o Scene Builder está concluída.




Instegração do Scene Builder com o IntelliJ

Passo 1: Ao abrir a interface do IntelliJ clique em "New Project".



Passo 2: Na barra esquerda selecione "JavaFX", dê um nome para o projeto, e deixe o restante das configurações como na imagem abaixo. Logo após, clique em "Next".



Passo 3: Selecione as bibliotecas que você usará e clique em "Create". Após isso você verá os arquivos gerados no seu projeto.



Passo 4: Clique com o botão direito do mouse no arquivo com a extensão ".fxml", procure e clique na opção "Open in SceneBuilder".



Passo 5: Na pasta onde o Scene Builder está instalado, selecione o arquivo "SceneBuilder.exe" e clique em "OK".



Passo 6: Após isso seu Scene Builder irá abrir mostrando a interface básica gerada pelo código do seu projeto no IntelliJ e as ferramentas de JavaFX estarão prontas para uso.





Instalação para Linux

Instalação do IntelliJ

Passo 1: Baixe o IntelliJ IDEA: Acesse o site oficial da JetBrains em https://www.jetbrains.com/idea/download/ e faça o download da versão "Community" ou "Ultimate" do IntelliJ IDEA para Linux.



Passo 2: Extraia o arquivo: Após o download, navegue até o local onde o arquivo foi baixado e extraia-o para uma pasta de sua escolha.



Passo 3:Inicie o IntelliJ IDEA: Navegue até a pasta onde o IntelliJ IDEA foi encontrado e encontre o arquivo "idea.sh". Abra um terminal nessa pasta e execute o seguinte comando para iniciar o IDE:

./idea.sh

Passo 4: Siga as instruções de configuração: Na primeira execução, o IntelliJ IDEA irá solicitar algumas configurações iniciais, como tema, plugins, etc. Siga as instruções vistas anteriormente para concluir a configuração inicial. Após isso o IDEA está pronto para uso


Instalação do Scene Builder


Passo 1:Baixe o Scene Builder: Acesse o site oficial do Gluon em https://gluonhq.com/products/scene-builder/ e faça o download do Scene Builder para Linux.


Passo 2:Instale o OpenJFX: O Scene Builder depende do OpenJFX. Verifique se você tem o OpenJFX instalado em seu sistema. Se não tiver, instale-o usando o gerenciador de pacotes da sua distribuição Linux. Por exemplo, no Ubuntu, você pode instalar o OpenJFX com o seguinte comando:

sudo apt-get install openjfx>

Passo 3:Extraia o arquivo: Após o download, navegue até o local onde o arquivo do Scene Builder foi baixado e extraia-o para uma pasta de sua escolha.



Passo 4:Inicie o Scene Builder: Navegue até a pasta onde o Scene Builder foi encontrado e encontre o arquivo "SceneBuilder". Abra um terminal nessa pasta e execute o seguinte comando para iniciar o Scene Builder:

./SceneBuilder

Integração do Scene Builder com o IntelliJ IDEA


Passo 1:Abra o Intellij e nstale o Plugin JavaFX: Se você não tiver o plugin JavaFX instalado no IntelliJ IDEA, vá para "Configurações" (Settings) -> "Plugins" -> "Marketplace" e procure por "JavaFX." Instale o plugin e reinicie o IntelliJ IDEA, se necessário.

Passo 2:Configure o Scene Builder no IntelliJ IDEA: Após instalar o plugin JavaFX, vá para "Configurações" (Settings) -> "Languages ​​& Frameworks" -> "JavaFX." No campo "Path to SceneBuilder," clique no botão de busca e navegue até o local onde você extraiu o Scene Builder. Selecione o executável "SceneBuilder" e clique em "OK".

Passo 3:Crie um novo projeto JavaFX: Crie um novo projeto JavaFX no IntelliJ IDEA ou abra um projeto existente.

Passo 4:Crie uma nova interface JavaFX: No IntelliJ IDEA, clique com o botão direito do mouse na pasta "src" do seu projeto, selecione "New" -> "JavaFX" -> "JavaFX FXML Document." Isso abrirá o Scene Builder com um arquivo FXML vazio.

Passo 5:Projete a interface do usuário no Scene Builder: No Scene Builder, você pode projetar sua interface do usuário usando os recursos fornecidos. Adicione componentes, ajuste as propriedades e organize o layout da sua aplicação..

Com isso a integração é concluida e você pode fazer seus projetos JavaFX da maneira que quiser.
